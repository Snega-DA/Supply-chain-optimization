# Supply-chain-optimization

## Overview

This project focuses on analyzing supply chain operations using Excel to uncover insights related to sales performance, supplier efficiency, inventory management, shipping costs, and product defects. The dashboard provides a centralized view of key business metrics to support data-driven decision-making and operational improvement.

## Objectives

* Monitor supply chain performance through interactive dashboards
* Analyze supplier efficiency and shipping costs
* Track inventory availability and product performance
* Identify trends, risks, and operational inefficiencies

## Tools & Technologies

* Microsoft Excel
* Power Query
* Pivot Tables & Pivot Charts
* Slicers & KPI Cards
* Data Cleaning & Transformation

## Key Features

* Interactive dashboard with dynamic filtering
* KPI tracking for sales, shipping cost, inventory, and supplier performance
* Trend analysis for operational metrics
* Automated data transformation using Power Query
* Visual insights for faster business reporting

## Insights Generated

* Identified suppliers with high defect rates and shipping costs
* Analyzed inventory trends and stock availability
* Tracked transportation performance across different routes
* Highlighted opportunities for cost optimization and process improvement

## Outcome

The project improved supply chain visibility by transforming raw operational data into meaningful business insights through an easy-to-use and interactive Excel dashboard.
